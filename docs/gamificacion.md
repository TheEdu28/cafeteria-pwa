# 6. Ludificacion (Gamificacion)

## Problema o comportamiento que atiende
El objetivo es motivar la recompra y la exploracion del menu, reduciendo la friccion en el proceso de pedido. La gamificacion busca incrementar la frecuencia de compra, incentivar el uso de categorias especificas (Desayunos) y reforzar la satisfaccion posterior a la compra con retroalimentacion positiva inmediata.

## Elementos de gamificacion integrados
- CafePuntos: puntos virtuales acumulables por monto de compra.
- Barra de progreso: indica el avance hacia una recompensa tangible (producto gratis).
- Micro-recompensas e insignias: logros por conducta deseada (frecuencia y categoria).
- Feedback inmediato: mensaje de puntos que se ganan antes de confirmar el pedido y puntos obtenidos tras confirmar.

## Como se reflejan en la interfaz
- Inicio (Home): tarjeta tipo dashboard con CafePuntos, progreso y listado de insignias desbloqueables.
- Menu y Pedidos: panel compacto con progreso actual para mantener el objetivo visible en cada visita.
- Confirmacion de pedido: mensaje claro de los puntos que se ganaran y puntos obtenidos una vez confirmado.

## Por que estos elementos mejoran la experiencia
- Claridad de objetivo: la barra de progreso y el contador de puntos hacen tangible la meta (producto gratis).
- Refuerzo positivo: el feedback inmediato aumenta la satisfaccion y refuerza el comportamiento de compra.
- Retencion: las insignias proponen metas secundarias que invitan a volver.
- Transparencia: el usuario sabe exactamente cuantos puntos gana y cuanto falta para la recompensa.
